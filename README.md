# Docker compose and laravel
```
1. docker-composer up
2. http://localhost/
3. Install Laravel with composer: docker-compose run --rm composer create-project laravel/laravel .
4. Install Laravel with composer: docker-compose run --rm composer i
5. Install a package with composer: docker-compose run --rm composer require spatie/laravel-permission
6. docker-compose run --rm composer chmod -R 777 storage
7. docker-compose run --rm npm i
```
